WerkStart – Burgerschap website
===============================
• Zwarte achtergrond (Tailwind) en paars‑blauwe gradient in het logo (WerkStart).
• Multi‑page structuur: index + 6 subpagina's.
• Templates voor CV, motivatiebrief en gesprek-checklist staan in assets/tools/.
• Start lokaal door index.html in je browser te openen (Tailwind via CDN).
